# Saudi Arabia Unemployment Analysis 2022-2023

Python data analysis project exploring unemployment trends in 10 Saudi regions using official statistics.

## Tools Used
- Python
- Pandas 
- Matplotlib
- OpenPyXL

## Key Findings
1. Overall decrease in unemployment rates between 2022-2023
2. Najran had the highest unemployment rate in 2023: 7.9%
3. Eastern Region had the lowest unemployment rate in 2023: 4.2%
